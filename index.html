Samsara

Global Game Jam 2016 game submission.

View the Project on GitHub tkesgar/ggj16-wayang

The cycle of birth, life, and death. A ritual of life to atone human's action in the past, present, and future. In this game, the tale is presented in unique style of "Wayang Kulit", a Javanese's traditional shadow puppet show.

How to Run

WebGL version of the game is available here. Alternatively, you can obtain the Windows version from the game's Global Game Jam 2016 page.
